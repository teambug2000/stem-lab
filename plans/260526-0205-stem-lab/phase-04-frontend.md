# Phase 04: Frontend Layout & Styling (CSS)
Status: ⬜ Pending
Dependencies: [Phase 01](file:///c:/Users/Admin/Desktop/stem/plans/260526-0205-stem-lab/phase-01-setup.md)

## Objective
Xây dựng giao diện Web App tĩnh và định dạng phong cách thiết kế hiện đại (Glassmorphism, Dark mode, CSS Variables). Đảm bảo giao diện hiển thị xuất sắc và mang lại cảm giác cao cấp (Premium Design).

## Requirements
- [ ] Thiết kế bảng Grid Lịch biểu thời gian thực hiển thị 4 Zone ở hàng ngang, các ca ở hàng dọc.
- [ ] Xây dựng Widget đổi vai trò (Role Switcher) nổi bật, dễ chuyển đổi.
- [ ] Định dạng 3 phân vùng hiển thị riêng biệt tùy thuộc vào vai trò đang chọn (Học sinh, Lab Assistant, Giáo viên).
- [ ] Tích hợp các modal: Form đăng ký phòng, Form đánh giá dịch vụ.
- [ ] Thiết kế phần chứa biểu đồ thống kê Chart.js trực quan.

## Implementation Steps
1. [ ] Cập nhật `index.html` với cấu trúc HTML5 đầy đủ các phân vùng giao diện: Header, Main Content (Student Grid, LA Dashboard, Teacher Dashboard), Modal Forms, và Widget Switcher.
2. [ ] Trong `styles.css`, thiết lập CSS Variables cho bảng màu hiện đại (Hủy các màu cơ bản thô cứng, dùng HSL/RGB mờ cho hiệu ứng kính).
3. [ ] Viết hiệu ứng Hover, Transition cho các ô lịch trống và các button.
4. [ ] Đảm bảo giao diện co giãn tốt trên màn hình máy tính và thiết bị di động (Responsive Layout).

## Files to Create/Modify
- [MODIFY] `index.html` - Thêm cấu trúc DOM cho các phân hệ giao diện
- [MODIFY] `styles.css` - Viết CSS định dạng phong cách cao cấp, hiện đại

## Test Criteria
- [ ] Giao diện hiển thị đúng layout mong muốn trên cả Desktop và giả lập Mobile.
- [ ] Không bị lỗi hiển thị phông chữ, khoảng cách cân đối, màu sắc hài hòa.
